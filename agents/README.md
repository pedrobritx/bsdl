# Agents — The Org Chart

This folder is the company's org chart. Each file defines a role an AI can assume.

```
CEO (Human)
 ├── product_manager
 ├── architect
 ├── frontend_engineer
 ├── backend_engineer
 ├── designer
 ├── qa_engineer
 ├── devops_engineer
 ├── security_engineer
 ├── data_engineer
 └── technical_writer
```

**Usage:** "Act as `agents/frontend_engineer.md`." The AI loads the role, its required reading, and its boundaries — no repetition needed.

Every agent file has the same six sections: Responsibilities · Priorities · Required reading · Never do · Definition of Done · Escalation rules.
